# Introduction

![](.gitbook/assets/nukeforbeginners01-min.png)

## 환영합니다.

* 영상 합성툴 소프트웨어중 뉴크\(Nuke\)라는 툴에대한 E-book 입니다.
* 뉴크의 기초 지식을 다루고 있습니다.
* 이 자료는 디지털아이디어의 산학연, 외부교육에 활용되고 있습니다.
* 첨부된 이미지는 macOS에서 스크린샷 되었습니다. 여러분이 윈도우즈를 사용한다면 스크린샷 이미지와 약간 다를 수 있습니다.
